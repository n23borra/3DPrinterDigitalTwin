# Frontend

Ce répertoire contient l'application React + Vite.

## Fichiers et dossiers
- `Dockerfile` : image Docker du frontend.
- `nginx.conf` : configuration Nginx pour le déploiement du frontend.
- `index.html` : point d'entrée HTML.
- `package.json` / `package-lock.json` : dépendances et scripts npm.
- `vite.config.js` : configuration Vite.
- `eslint.config.js` : configuration ESLint.
- `postcss.config.js` / `tailwind.config.js` : configuration PostCSS/Tailwind CSS.
- `public/` : assets statiques publics.
- `src/` : code source React.
