# Composants

## Fichiers
- `Button.jsx` : bouton réutilisable.
- `Loader.jsx` : indicateur de chargement.
- `LogoutButton.jsx` : bouton de déconnexion.
- `Modal.jsx` : composant de fenêtre modale.
- `PrivilegedRoute.jsx` : route protégée pour rôles privilégiés.
- `ProtectedRoute.jsx` : route nécessitant authentification.
- `Sidebar.jsx` : barre latérale de navigation.
- `Sidebar.module.css` : styles de la barre latérale.
- `SuperAdminRoute.jsx` : route protégée pour super-admin.
- `TagInput.jsx` : saisie d'étiquettes.

## Sous-dossier
- `printers/` : composants spécifiques aux imprimantes.
