# Backend

Ce répertoire contient l'application Spring Boot.

## Fichiers et dossiers
- `Dockerfile` : image Docker du backend.
- `pom.xml` : configuration Maven et dépendances.
- `mvnw`, `mvnw.cmd` : wrappers Maven pour exécuter Maven sans installation locale.
- `src/main/` : code source principal de l'application.
- `src/test/` : tests automatisés.
