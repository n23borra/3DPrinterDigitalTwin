# Package com.fablab.backend.controllers

## Fichiers
- `AccountController.java` : endpoints liés au compte utilisateur.
- `AuthController.java` : endpoints d'authentification.
- `AuditController.java` : endpoints des journaux d'audit.
- `UserController.java` : endpoints de gestion des utilisateurs.

## Sous-dossier
- `printer/` : endpoints liés aux imprimantes.
